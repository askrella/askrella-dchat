# Askrella Decentralized Chat

-   React
-   Vite
-   Tailwind
-   React Router
-   [Zustand](https://github.com/pmndrs/zustand) for State Management

## Available Scripts

| Name            | Description                                          |
| --------------- | ---------------------------------------------------- |
| `npm run dev`   | Runs the app in the development mode.                |
| `npm run build` | Builds the app for production to the `build` folder. |
